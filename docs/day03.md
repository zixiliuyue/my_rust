# Day 3：函数、控制流、结构体和方法

## 先运行

```bash
cargo run --bin day03_structs
```

## 再读代码

重点阅读 `src/bin/day03_structs.rs`：

- `struct User` 把姓名和年龄组织成一个类型。
- `impl User` 给类型添加关联函数和方法。
- `Self` 表示当前类型。
- `&self` 表示方法只读借用当前对象。

## 底层原理

结构体不是运行时字典，它的字段布局在编译期已知。像 `u32` 这样的固定大小字段可以直接内联在结构体中；像 `String` 这样的字段会把指针、长度、容量放在结构体里，堆数据仍在堆上。

方法调用 `user.say_hello()` 会被编译器转换成带 `self` 参数的函数调用。Rust 的方法语法主要提高可读性，不代表对象一定有运行时继承结构。

## 练习

- 给 `Rectangle` 添加 `perimeter()` 方法。
- 新增一个 `Circle` 结构体，包含半径并计算面积。
- 把 `User::new` 的年龄限制为小于 150，超出时返回 `None`。

## 常见坑

- 函数最后一行没有分号才会作为返回值。
- `self`、`&self`、`&mut self` 的所有权语义不同。
- `String` 字段会移动所有权，传参前要确认后面是否还要继续使用原变量。
